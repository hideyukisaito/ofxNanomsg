ofxNanomsg
===

openFrameworks addon for [nanomsg](http://nanomsg.org).  
The API is almost compatible with [ofxZmq](http://github.com/satoruhiga/ofxZmq).

**Important:**  
**nanomsg is still in alpha, so implementation of this addon will be changed.**
